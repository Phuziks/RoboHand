# Status Web Application

## Описание
Простое веб-приложение на Flask, которое отображает текущий статус (ON/OFF) и позволяет переключать его с помощью ссылок.

## Функциональность
- Отображение текущего статуса: **ON** или **OFF**
- Переключение статуса через ссылки:
  - `Включить (ON)` - устанавливает статус "ON"
  - `Выключить (OFF)` - устанавливает статус "OFF"
- При первом запуске или при отсутствии параметра статус по умолчанию - **OFF**

## Установка и запуск

### Требования
- Python 3.x
- Flask

### Установка зависимостей
```bash
pip install flask
```

### Запуск приложения
```bash
python app.py
```

После запуска приложение будет доступно по адресу: `http://localhost:5000`

## Использование
1. Откройте браузер и перейдите по адресу `http://localhost:5000`
2. Страница отобразит текущий статус
3. Нажмите на ссылку "Включить (ON)" для установки статуса ON
4. Нажмите на ссылку "Выключить (OFF)" для установки статуса OFF

## Параметры URL
Приложение поддерживает GET-параметр `status`:
- `?status=on` - установить статус ON
- `?status=off` - установить статус OFF

Примеры:
- `http://localhost:5000/?status=on` - отобразит статус ON
- `http://localhost:5000/?status=off` - отобразит статус OFF

## Технические детали
- **Фреймворк:** Flask
- **Порт:** 5000
- **Хост:** 0.0.0.0 (доступно с других устройств в сети)
- **Режим:** Debug (включен для разработки)
